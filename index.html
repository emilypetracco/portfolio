A React-based trivia app that utilises the Open Trivia Database API and Firebase. Users can select a category, number of questions and the difficulty of the questions and take a trivia quiz. Cocktail Companion

A React-based app that helps users find the perfect drink, data is gathered using the Cocktail Database API. Users can decide if they wish to randomly generate cocktails or shots at the click of a button. Upon each click, a randomised drink will appear, complete with an image, ingredients list and instructions on how to make it yourself.

Makeup Your Mind

A jQuery app that helps users find new and exciting makeup products. Users can select what type of product they're looking for and filter out products with specific requirements. The products are printed to the page with the product name and brand, an image and a link to buy the item. Data is gathered using the Makeup API. This project was built using pair-programming with Olivia Heulitt.
